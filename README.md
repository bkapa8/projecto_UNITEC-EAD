## Parte técnica  (blabbla bla)

A stack usada para a concepção da plataforma incluem: Next.JS, Clerk e PostgreSQL. 

Portanto, lógicamente que para trabalhares com meu projecto no  PC, precisas ter node.js(pô, nem precisava dizer...):


```bash

git clone https://github.com/bkapa8/projecto_UNITEC-EAD.git   
cd projecto_UNITEC-EAD   
npm install  

# aqui vais se dar mal, kkk 
cp .env.example .env.local

npm run db:push
npm run dev
```
